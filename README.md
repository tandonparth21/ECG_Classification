ECG Image Classification Using CNN, SVM, XGBoost, AdaBoost, and R

Developed a robust hybrid ECG image classification system combining deep learning (Convolutional Neural Networks) with traditional machine learning algorithms (SVM, XGBoost, AdaBoost) to accurately detect and classify cardiac conditions.

Preprocessed ECG images and extracted relevant features to enhance model performance and generalization.

Trained and fine-tuned multiple models, evaluated their effectiveness using standard metrics (accuracy, precision, recall, F1-score), and selected optimal classifiers based on comparative results.

Integrated the classification results within R to create interactive and dynamic visualizations, enabling intuitive exploration of model performance and diagnostic insights.

Conducted thorough comparative analysis of different models, providing actionable insights on their strengths and limitations in ECG diagnosis.
